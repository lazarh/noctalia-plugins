# Daylight

A simple bar widget that displays sunrise, sunset times and daylight duration.

## Features

- Shows sun icon during day, moon icon at night
- Hover to see sunrise time, sunset time, and total daylight duration
- Click to configure location
- Uses location-based weather data for accurate times

## Configuration

- **Location**: Set your location in "city, country" format (e.g., "London, UK")

## Requirements

- Noctalia 4.3.0 or higher
- Location service enabled for weather data
